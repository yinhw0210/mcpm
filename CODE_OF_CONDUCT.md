# Code of conduct

MCP Manager aims to be a calm, practical project for people building with MCP tools.

## Expected behavior

- Be respectful and direct
- Assume good intent, but ask for clarification when behavior is unclear
- Keep feedback focused on the work
- Share enough context for others to reproduce issues
- Redact secrets from logs, screenshots, and config snippets

## Unacceptable behavior

- Harassment, threats, or personal attacks
- Publishing private information without consent
- Repeated off-topic comments after maintainers redirect the discussion
- Sharing credentials, tokens, or private configuration data from another person or organization

## Enforcement

Maintainers may edit, hide, or remove comments, issues, and pull requests that violate this code. Maintainers may also temporarily or permanently limit participation when behavior puts contributors or users at risk.

Report concerns to the maintainer through GitHub.
