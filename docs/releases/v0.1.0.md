# MCPM v0.1.0

MCPM 的第一个公开版本，面向同时使用多个 AI 编程工具的用户。它把分散在不同编辑器、CLI 和桌面应用里的 MCP Server 配置收进一个可视化桌面应用，让安装、查看、同步和调试 MCP Server 变得更清晰。

This is the first public release of MCPM, a desktop control center for managing MCP servers across local AI coding agents.

## Highlights

- Manage MCP servers across 25 agent targets, including Claude Code, Codex, Cursor, VS Code, Zed, Goose, Kiro, OpenCode, Windsurf, Gemini CLI, Claude Desktop, Cline, and more.
- Add an MCP server once from a URL, npm package, command, or local executable, then write the native config format for selected agents.
- Switch between service-first and agent-first views to understand where each MCP server is enabled.
- Search both the add-mcp registry and the official MCP registry, with metadata prefill for install flows.
- Debug stdio MCP servers with managed sessions, live stdout/stderr logs, CPU and memory metrics, and JSON-RPC input.
- Detect local runtimes such as Node.js, `npx`, Python, Bun, and `uvx`.
- Use the app in either English or Simplified Chinese.

## Built on add-mcp

MCPM is a desktop-oriented second development inspired by and based on ideas from [neon-solutions/add-mcp](https://github.com/neon-solutions/add-mcp). The upstream project provides excellent command-line workflows for adding, finding, removing, and synchronizing MCP server configs. MCPM brings those ideas into a Tauri desktop app and adds visual management, bilingual UI, runtime detection, process debugging, remote status checks, and tray behavior.

## Download

Download the installer for your platform from the release assets:

- macOS: download the `.dmg` package.
- Windows: download the `.msi` or `-setup.exe` installer.

If a platform asset is not attached yet, build from source:

```bash
pnpm install
pnpm tauri build
```

## Notes

- This is an early desktop release. Please review generated project-level MCP config before committing it to your own repositories.
- MCPM can start local stdio server commands during debugging. Only run MCP servers you trust.
- macOS and Windows may show extra security prompts if the installer is not code-signed yet.

## Verification

Before publishing this release, run:

```bash
pnpm run check:copy
pnpm run typecheck
pnpm run build
cd src-tauri
cargo test
```
